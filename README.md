# Installation guide
## Clone this repo:

```shell
git clone https://github.com/wh1tePigeon/AutoDub
```

## Install requirements:

```shell
pip install -r requirements.txt
```
## Download necessary checkpoints:

```shell
python download_checkpoints.py
```
